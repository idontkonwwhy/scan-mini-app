# 五金配件目录 — 设计文档

**日期:** 2026-06-18  
**类型:** 纯 H5 移动端  
**设计风格:** Apple 轻量简洁

---

## 概述

手机扫码即看的五金配件产品目录。三个页面：主页（分类浏览 + 搜索）、商品详情、联系名片。

## 技术栈

- Vite + Vue3 + Vue Router（hash 模式）
- 纯静态部署，数据用 JSON 文件，图片放本地
- 移动端优先，vw 单位适配

## 数据模型

### 分类
```json
{
  "id": "c1",
  "name": "铰链",
  "order": 1
}
```

### 商品
同一商品多个尺寸/型号 → variants 数组，每个 variant 渲染为独立卡片。
```json
{
  "id": "p1",
  "categoryId": "c1",
  "name": "不锈钢液压铰链",
  "image": "/images/p1.jpg",
  "desc": "不锈钢材质，液压缓冲，开合静音。适用于衣柜、橱柜门板，安装简便，使用寿命长。",
  "variants": [
    { "id": "v1", "label": "A-001", "spec": "100×60mm" },
    { "id": "v2", "label": "A-002", "spec": "90×50mm" }
  ]
}
```

## 页面设计

### 主页 (Home.vue)

**布局:** 毛玻璃顶栏 → 横向滑动 Chlip 分类 → 两列商品网格 → 底部 Tab 栏

- **顶栏:** 标题"五金目录" + 搜索图标（圆形按钮，点击展开搜索框）
- **分类 Chips:** 横向滑动，选中的黑底白字，其余白底黑字
- **商品卡片:** 圆角 14px，极轻阴影。上方正方形图片区，下方两行（商品名 + 型号/尺寸）。variant 各自独立卡片
- **搜索:** 展开后输入即过滤商品名，大小写不敏感
- **底部 Tab:** 首页 / 联系，毛玻璃背景

**配色:** 背景 #F5F5F7，卡片 #FFF，选中 #1D1D1F，文字 #1D1D1F / #8E8E93

### 详情页 (Detail.vue)

**布局:** 返回键 → 商品大图 → 名称 + 型号 + 简介文字

- 上方 1:1 正方形商品大图
- 下方简单信息区：商品名（19px 加粗）、型号规格（14px 灰色）、简介（14px）
- 底部留白，保持呼吸感
- 无底部 Tab 栏（返回首页通过左上角返回）

### 联系页 (Contact.vue)

**布局:** 顶栏 → 名片卡片

- 名片卡片：公司名 + 一句话定位 + 电话 + 地址 + 微信二维码
- 信息前有 emoji 图标（📞📍）
- 二维码占位区（后期替换为真实二维码图片）
- 底部 Tab：首页 / 联系（联系高亮）

## 路由

```
/           → Home.vue     # 主页
/detail/:productId/:variantId → Detail.vue  # 商品详情（variantId 可选）
/contact    → Contact.vue  # 联系页
```

## 静态资源结构

```
public/images/
  p1.jpg     # 商品主图，文件名与 products.json 中的 image 字段对应
  qr.jpg     # 微信二维码
src/data/
  categories.json
  products.json
```

## 交互细节

- 搜索实时过滤，无需按确认键
- 点击分类 Chip 切换分类，同时清空搜索词
- 切换分类或搜索后，网格内容即时更新
- 点击卡片进入详情页
- 底部 Tab 切换首页/联系，保持激活态高亮
- 所有过渡用 CSS transition 180ms ease-out

## 不做

- 无用户登录/注册
- 无购物车、下单
- 无后台管理（数据手动编辑 JSON）
- 无 PWA（按需后续加）
- 无多语言
